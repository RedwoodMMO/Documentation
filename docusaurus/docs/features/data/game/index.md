# Game Data

Redwood provides auto-synced [player](./player-data.md) and [world](./world-data.md) game data. The included Unreal plugins will receive the JSON data returned by the backend API and deserialize that data into Unreal Blueprint or C++ structs.

Redwood also supports generic [blob/document storage](../storage/blob-storage.md) if you're interested in storing images, PDFs, SaveGames, etc.
